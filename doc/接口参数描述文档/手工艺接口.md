## /수공예 인터페이스/사용자 상호 작용/사용자 로그인

```text
- 사용자 로그인 인터페이스
    - 선행 작업
        - 인증 코드 인터페이스 호출 필요，현재 인증 코드 그림에 해당하는 키 값을 가져옵니다，나중에 로그인 인터페이스를 호출할 때 현재 키를 가져가야 합니다.
    - 후면 작업
        - token, userinfo, exp 정보를 가져옵니다.
        - 토큰은 인터페이스를 호출할 때마다 헤더에 넣어야 합니다. 키는"authorization"이고 값은 이 토큰의 값입니다.
        - userinfo에는 사용자 id가 들어 있습니다. 나중에 사용자 작업을 할 때 현재 사용자 id를 사용하여 작업을 수행할 수 있습니다.
        - exp 는 현재 로그인 상태의 만료 시간을 나타내며 기본값은 하루입니다.
```
